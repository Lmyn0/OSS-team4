# 오픈소스소프트웨어 팀 프로젝트
 - 난이도 선택 기능과 Kruskal 알고리즘을 이용한 미로 생성 게임

# 프로젝트 개요
 - 이 프로그램은 Kruskal 알고리즘을 이용해 매번 랜덤한 미로를 생성하고, 플레이어가 방향키를 사용해 출발점에서 도착점까지 이동해야 하는 간단한 2D 미로 게임입니다.
 - 난이도에 따라 미로 크기와 셀 크기가 달라집니다.

# 주요 기능
 - Kruskal 알고리즘을 이용한 미로 생성
    - 각 셀을 독립된 집합으로 보고, 임의의 벽을 허물어 연결해서 미로를 생성
    - 사이클이 생기지 않도록 Union-Find 자료구조로 관리
    - 벽이 모두 처리되면 안전한 미로 완성
- 플레이어 이동
    - Player 클래스는 현재 위치(x, y)를 저장하고 각 방향(N, S, E, W)에 벽이 없을 경우에만 이동 가능
- 난이도 조절
    - difficulty.py의 select_difficulty() 함수를 통해 쉬움/어려움 난이도 선택
    - 쉬움 : 20 x 20 셀, 셀 크기 60px
    - 어려움 : 30 x 30 셀, 셀 크기 50px
